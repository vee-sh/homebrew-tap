# Homebrew Tap for veessh

This tap contains the Homebrew formula for [veessh](https://github.com/vee-sh/veessh).

## Installation

```bash
brew tap vee-sh/tap
brew install veessh
```

## Updating

```bash
brew update
brew upgrade veessh
```

## About veessh

veessh is a console connection manager supporting SSH, SFTP, Telnet, Mosh, AWS SSM, and GCP gcloud.

For more information, see the [main repository](https://github.com/vee-sh/veessh).
